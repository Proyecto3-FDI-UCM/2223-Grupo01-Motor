\doxysubsection{Detailed Description}
Manages a SDL\+\_\+\+Window instance so it can be shared among several modules like \mbox{\hyperlink{namespace_ogre}{Ogre}} and Bullet. By default, it\textquotesingle{}ll initialize SDL and create an 854x480 SDL window named \char`\"{}\+Motor Engine\char`\"{}, and will take input focus. Initialize the window with\+: Window\+::init(\+Uint32 sdl\+Flags, const char$\ast$ name, int x, int y, int w, int h, Uint32 window\+Flags); and access the window with win().\mbox{\hyperlink{classme_1_1_window_a7aab48895f33d6a5ecc2f5a1c0e582c7}{get()}};
