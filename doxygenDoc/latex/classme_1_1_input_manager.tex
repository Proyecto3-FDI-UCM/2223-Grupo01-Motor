\doxysubsection{Detailed Description}
\mbox{\hyperlink{classme_1_1_input_manager}{Input\+Manager}} provides information and callbacks for any user input from keyboard, mouse and game controller. Define virtual buttons to easely remap your game\textquotesingle{}s controls. You can access the \mbox{\hyperlink{classme_1_1_input_manager}{Input\+Manager}} just calling im().
