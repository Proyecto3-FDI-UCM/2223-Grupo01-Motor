\doxysubsection{Detailed Description}
Default implementation of a Listener to use with the \mbox{\hyperlink{namespace_ogre_1_1_r_t_shader}{Ogre\+::\+RTShader}} system. When a target scheme callback is invoked with the shader generator scheme it tries to create an equivalent shader based technique based on the default technique of the given material.

This is the hook point where shader based technique will be created. It will be called whenever the material manager won\textquotesingle{}t find appropriate technique that satisfy the target scheme name. If the scheme name is out target RT Shader System scheme name we will try to create shader generated technique for it.
