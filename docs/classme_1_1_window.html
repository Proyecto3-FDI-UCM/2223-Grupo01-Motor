Detailed Description

Manages a SDL_Window instance so it can be shared among several modules like Ogre and Bullet. By default, it'll initialize SDL and create an 854x480 SDL window named "Motor Engine", and will take input focus. Initialize the window with: Window::init(Uint32 sdlFlags, const char* name, int x, int y, int w, int h, Uint32 windowFlags); and access the window with win().get();

Parameters
sdlFlags0, or one or more SDL_InitFlags OR'd together
namename of the window in UTF-8 encoding.
x,yposition of the window. Use SDL_WINDOWPOS_CENTERED or SDL_WINDOWPOS_UNDEFINED.
w,hwidth and height of the window in screen coordinates
windowFlags0, or one or more SDL_WindowFlags OR'd together
