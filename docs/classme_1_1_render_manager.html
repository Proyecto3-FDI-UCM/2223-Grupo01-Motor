Detailed Description

OgreManager initialize Ogre (Root, RTShaderSystem, SceneManager, RenderWindow), locate and load resources (from resource.cfg) and handle the creation of camera, light, mesh You can access the OgreManager just calling om().

◆ mOverlaySystem

Ogre::OverlaySystem* me::RenderManager::mOverlaySystem
protected

This class simplify initialization / finalization of the overlay system. Before you create a concrete instance of the OverlaySystem the OGRE::Root must be created but not initialized. In the ctor all relevant systems are created and registered. The dtor must be called before you delete OGRE::Root. To make the overlays visible (= render into your viewports) you have to register this instance as a RenderQueueListener in your scenemanager(s).
